The ``MyList`` class
====================

How to use the MyList class:
    >>> MyList = __import__('1-my_list').MyList

Normal use:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Negative numbers:
    >>> my_list = MyList([1, -5, 10, -2])
    >>> my_list.print_sorted()
    [-5, -2, 1, 10]

Non-integer (though the prompt assumes integers, good to test):
    >>> my_list = MyList(["b", "a", "c"])
    >>> my_list.print_sorted()
    ['a', 'b', 'c']
